NSS Project: War

A rendition of the classic card game using the http://deckofcardsapi.com/ api for an NSS class project. The game involves each player recieving a seperate deck of cards, and the winner is declared after 52 cards are gone.

Technologies

  1. Bootstrap
  2. RequireJS
  3. Bower
  4. Grunt
  5. SASS
  6. Q Promises

Requirements:

  1. Nodejs
  2. Bower

Installation Instructions

  1. git clone https://github.com/dan-hodges/WAR.git && cd WAR/lib
  2. npm install May take a few minutes to download all dependencies, also runs bower install.
  3. npm start This fires up the web server
